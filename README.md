# Cybersecurity-Presentations

Here you can find my presentations on cybersecurity, which are for educational purposes.
1. Intro to Cybersecurity and Useful Tools
2. Cybersecurity Threats and Trends
3. Cybersecurity for Regular Users

At the end of the first presentation you can find a link with a quiz to test your understanding on the topic.

[Quiz 1](https://docs.google.com/forms/d/e/1FAIpQLSdRV3Ee46TrsiJVf4mGn8AO2lDnd8yNbbSe0I2SsOZ48UXpVQ/viewform)

These presentations do not follow the typical guidelines (less text, more images), as they are not intended to be presented but to be read asynchronously. My main goals were to fully understand the content myself, disseminate the work I carried out as a cybersecurity intern, develop an effective presentation structure and respect copyright requirements. Because of the latter, I generally avoided using many images.
